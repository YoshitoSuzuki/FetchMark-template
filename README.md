# FetchMark (iOS App)

[English](./docs/en/README.md) | [日本語](./docs/jp/README.md)

FetchMark is a beautiful viewer for Markdown files hosted on the web (e.g., GitHub Pages). It preserves your directory structure and provides a clean, native-like experience for reading your documentation or personal notes on the go.

## Main Features
- **Hierarchical Navigation**: Preserves folder structures defined in your repository.
- **Custom Styling**: Supports local (`style.css`) and global CSS for full layout control.
- **Auto-Update**: Built-in refresh mechanism to sync with your web content.
- **Automation Ready**: Provides scripts and GitHub Actions templates to sync your file index automatically.
